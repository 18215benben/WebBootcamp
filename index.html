Web Development Bootcamp for Kids

Easter Break • Ages 8–11 • Live Online

👋 What Is This?

A friendly, live web development bootcamp designed for children aged 8–11. During Easter Break, children learn how real websites are made using HTML, CSS, and simple interactive features.

By the end of the week, every child finishes their own website and feels confident explaining how it works.

🧑‍🏫 Who Is Teaching?

The sessions are taught by an 11-year-old web developer(which has been coding for 3 years) and is skilled at building websites and explaining coding clearly to other children. Here are some examples of websites he has built:

  • Rami Benbenishty - A personal portfolio website (in development)
  • Sandbox game - A simple game built using HTML, CSS, and JavaScript (Complete)
  • QR Code Generator - A simple QR code generator built using HTML, CSS, and JavaScript

Being close in age helps lessons feel relaxed, engaging, and easy to understand, especially for beginners.

✔ Parent communication is always available

✔ Small groups ensure a safe, focused environment

📅 How the Week Works

Lessons are progressive, meaning skills build day by day. Children may join with different experience levels, and everyone finishes together. Lessons take place at 10:00am GMT

Day Focus
Monday Foundations – What websites are & basic HTML
Tuesday Design – Styling with CSS
Wednesday Interactivity – Buttons & simple effects
Thursday Build – Combining everything
Friday Showcase – Finish & present projects
✔ Children with experience may join later in the week
✔ Missing a day is fine — catching up is easy

⏰ Lesson Format

Each session is 60 minutes and includes:

  • Clear explanation
  • Guided building time
  • Extra challenges for confident learners
  • Friendly, patient support
